## API Key Validation Logic

```python
# In helpers.py

def validate_api_key(provided_key: str) -> bool:
    """
    Validate the API key from request header
    
    Args:
        provided_key: The key from x-api-key header
        
    Returns:
        True if valid, False otherwise
    """
    # Get expected key from environment
    expected_key = settings.API_SECRET_KEY  # From .env file
    
    # Simple string comparison
    return provided_key == expected_key
```

1. **Storage:** API key stored in `.env` file â†’ loaded into `settings.py`
2. **Extraction:** Django extracts `x-api-key` from request headers
3. **Comparison:** Simple string equality check
4. **Result:** 
   - âœ… Match â†’ Continue processing
